# Tapocalypse — バックログ

Phase 1 のスコープ外アイデアはここに登録する。
実装する場合は PM が Phase を決定してから着手する。

フォーマット:
```
- [ ] [Phase N] 機能名: 概要（提案日: YYYY-MM-DD）
```

---

## Phase 2 — ゲーム化

- [ ] [Phase 2] ステージ制: ステージごとに難易度・時間・背景が変化する
- [ ] [Phase 2] 時間延長: 1000 タップ達成で残り時間 +3 秒
- [ ] [Phase 2] 背景変化: スコアに応じて宇宙背景がアニメーションする
- [ ] [Phase 2] ボーナスターゲット: ランダム位置に ⭐ が出現し +50 点

## Phase 3 — モード実装

- [ ] [Phase 3] Normal モード: 現在の Phase 1 ゲームプレイを正式化
- [ ] [Phase 3] Hard モード: 属性システム（火 / 水 / 雷）を追加
- [ ] [Phase 3] Hell モード: 長押しゾーン + 属性
- [ ] [Phase 3] Apocalypse モード: 移動ターゲット + 属性 + 長押し

## Phase 4 — 世界ランキング

- [ ] [Phase 4] オンラインランキング: スコアをサーバーに送信・ランキング表示
- [ ] [Phase 4] ニックネーム登録: プレイヤー名を設定してランキングに表示
- [ ] [Phase 4] 順位表示: 自分の世界順位をゲーム後に表示

## Phase 5 — SNS 拡散

- [ ] [Phase 5] シェア機能: 結果画面から X / Instagram に共有
- [ ] [Phase 5] ランキング共有: 「世界 XX 位！」の画像生成
- [ ] [Phase 5] 挑戦導線: 友達を挑戦に誘うリンク生成

## Phase 6 — 認定ランカー

- [ ] [Phase 6] 動画提出フォーム: プレイ動画をアップロードする仕組み
- [ ] [Phase 6] Verified Tapper バッジ: 審査通過者にバッジ付与
- [ ] [Phase 6] TOP10 認定ページ: 認定ランカー一覧ページ

## Phase 7 — Android 正式版

- [ ] [Phase 7] Capacitor 導入: Web → Android アプリ変換
- [ ] [Phase 7] Google Play 公開: ストア申請・公開
